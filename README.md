# 🎵 Project Overview

This project is a **Spotify Web Player UI Clone** built using **HTML and CSS only**.

The goal of this project is to **replicate the layout and user interface of Spotify's web player**, focusing on modern UI design and responsive layout techniques.

⚠️ Note:  
This project is **only the frontend interface**.  
No backend or music streaming functionality is implemented.

---

# 🖥 Interface Layout

The application is divided into **three main sections**.

```
Sidebar
   │
   ├── Home
   ├── Search
   └── Library

Main Content
   │
   ├── Recently Played
   ├── Trending Songs
   └── Featured Charts

Music Player
   │
   ├── Album Cover
   ├── Song Information
   ├── Playback Controls
   └── Volume Control
```

# 🎨 UI Components

### 🧭 Sidebar Navigation

Contains:

- Home
- Search
- Your Library
- Playlist creation
- Podcast recommendations

Built using **flexbox for vertical layout**.

---

### 🎼 Music Cards

Displays:

- Album cover
- Song title
- Artist information

Sections include:

- Recently Played
- Trending Now
- Featured Charts

---

### 🎧 Music Player

The bottom music player includes:

- Album artwork
- Song title and artist
- Play / Pause / Next / Previous controls
- Playback progress bar
- Volume slider

The player is **fixed at the bottom of the screen**.

# 📱 Responsive Design

The layout adapts for smaller screens.

Example media query used:

```
@media (max-width: 1000px)
```

Certain navigation elements hide automatically for better mobile display.
